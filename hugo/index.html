1.1 安装hugo

1.2 创建网站

1

hugo new site 路径

1.3 添加主题

  • 将主题直接添加到theme文件下面
  • 将主题作为一个submodule

1.4 创建文档

1

1

目录结构说明 以下是每个目录的高级概述，其中包含指向 Hugo 文档中每个相应部分的链接。

2.1 archetypes

hugo模板,在创建文件时作为模板自动生成

2.2 assets

存储所有需要HugoPipes处理的文件;只有使用了.Permalink 或 .RelPermalink的文件才会发布到公共目录. 注意：默认情况下不创建该目录

2.3 config

Hugo配置目录

2.4 content

此目录存在所有的网站内容,Hugo中的每个顶级文件夹都被视为一个内容部分.

2.5 data

该目录用于存储 Hugo 在生成网站时可以使用的配置文件

2.6 layouts

以 .html文件的形式存储模板.

2.7 static

存储所有静态内容:图像、CSS、JavaScript等。当Hugo构建您的站点时,静态目录中的所有资产都按原样复制
