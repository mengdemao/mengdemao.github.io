STL学习笔记

Oct 6, 2021 2 分钟阅读

STL称为标准模板库(Standard Template Library) 广义上可以分为容器,算法,迭代器 容器和算法通过迭代器进行无缝连接 STL几乎所有的代码都采用了函数模版或者类模板

STL组件

序号 名称 解释
1 容器 各种数据结构
2 算法 各种常用的算法
3 迭代器 容器域算法的胶合
4 仿函数 行为类似函数
5 适配器 修饰容器或者仿函数迭代器
6 空间配置器 负责空间的配置和管理

迭代器使用

迭代器方案1
1vector<int>::iterator itBegin = v.begin();
2vector<int>::iterator itEnd   = v.end();
3while (itBegin != itEnd) {
4	cout << *itBegin << endl;
5	itBegin += 1;
6}
迭代器2
1for (vector<int>::iterator it = v.begin(); it != v.end(); it++)
2{
3	cout << *it << endl;
4}
遍历算法
1template <class T>
2void myPrint(T val)
3{
4	cout << val << endl;
5}
6
7/* 可惜回调函数不支持自动推导 */
8for_each(v.begin(), v.end(), myPrint<int>);

容器自定义数据

容器嵌套容器

 1vector<vector<int>>v; // 外部大容器
 2vector<int> vx[10];   // 内部小容器
 3
 4/* 插入容器 */
 5for (int i = 0; i < 10; i++)
 6{
 7	for (int j = 0; j < 30; j++)
 8	{
 9		vx[i].push_back(i + j + 10);
10	}
11	v.push_back(vx[i]);
12}
13
14/* 遍历容器 */
15for (vector<vector<int>>::iterator it = v.begin(); it != v.end(); it++)
16{
17	for (vector<int>::iterator vit = it->begin(); vit != it->end(); vit++)
18	{
19		cout << *vit << " ";
20	}
21	cout << endl;
22}
